IgeSmartTexture CLASS

Declared in IgeSmartTexture.js

Description
Creates a new texture from a smart texture object.
Extends
IgeEventingClass

Methods

id(String id)

Gets / sets the current object id. If no id is currently assigned and no id is passed to the method, it will automatically generate and assign a new id as a 16 character hexadecimal value typed as a string.
  • {String}id
Returns {*} Returns this when setting the value or the current value if none is specified.

assignSmartTextureImage(String scriptObj)

Assigns a render script to the smart texture.
  • {String}scriptObj The script object.

_setImage(Image imageElement)

Sets the image element that the IgeTexture will use when rendering. This is a special method not designed to be called directly by any game code and is used specifically when assigning an existing canvas element to an IgeTexture.
  • {Image}imageElement The canvas / image to use as the image data for the IgeTexture.

textureFromCell(Number indexOrId)

Creates a new texture from a cell in the existing texture and returns the new texture.
  • {Number}indexOrId The cell index or id to use.
Returns {*}

sizeX(Number val)

Sets the _sizeX property.
  • {Number}val

sizeY(Number val)

Sets the _sizeY property.
  • {Number}val

resize(Number x, Number y, Boolean dontDraw)

Resizes the original texture image to a new size. This alters the image that the texture renders so all entities that use this texture will output the newly resized version of the image.
  • {Number}x The new width.
  • {Number}y The new height.
  • {Boolean}dontDraw If true the resized image will not be drawn to the texture canvas. Useful for just resizing the texture canvas and not the output image. Use in conjunction with the applyFilter() and preFilter() methods.

restoreOriginal()

Sets the texture image back to the original image that the texture first loaded. Useful if you have applied filters or resized the image and now want to revert back to the original.

render(CanvasContext2d ctx, IgeEntity entity)

Renders the texture image to the passed canvas context.
  • {CanvasContext2d}ctx The canvas context to draw to.
  • {IgeEntity}entity The entity that this texture is being drawn for.

preFilter( method)

Gets / sets the pre-filter method that will be called before the texture is rendered and will allow you to modify the texture image before rendering each tick.
  • {}method
Returns {*}

applyFilter(Function method, Object data)

Applies a filter to the texture. The filter is a method that will take the canvas, context and originalImage parameters and then use context calls to alter / paint the context with the texture and any filter / adjustments that you want to apply.
  • {Function}method
  • {Object}data
Returns {*}

stringify()
